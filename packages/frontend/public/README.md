# Pasta Public - Assets Estáticos

Esta pasta contém arquivos estáticos que serão servidos diretamente pelo Vite/Netlify.

## Logo

**Arquivo necessário:** `logo.png`

- Baixe o arquivo "Guru do dindin fundo transparente.png" do cPanel
- Renomeie para `logo.png`
- Adicione nesta pasta: `/packages/frontend/public/logo.png`
- Faça commit e push

O logo será acessível em: `https://gurudodindin.com.br/logo.png`

**Locais onde o logo é usado:**
- Layout.tsx (cabeçalho principal)
- Login.tsx (página de login)
- Register.tsx (página de registro)
